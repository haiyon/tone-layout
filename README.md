# Sample layout

> 示例服务，新建服务时以此为模板

## Layout

| 目录（模块）     | 描述                               |
|------------|----------------------------------|
| bin        | 生成的可执行文件                         |
| biz        | 业务逻辑处理、含不同服务之间逻辑、聚合等             |
| cmd/sample | 应用运行入口                           |
| cmd/xxx    | 根据情况使用，定义其它运行入口                  |
| conf       | 运行配置，通过 ptotoc 生成，对应 config.yaml |
| data       | 数据操作，对接数据库、缓存、搜索引擎、连接其它服务取数据等    |
| dto        | 结构映射，方便统一修改                      |
| server     | 服务运行时，http / gRPC 模式，对应 proto 定义 |
| service    | 服务入口，对接 proto 接口定义及业务逻辑层         |
|            |                                  |
